1. Under `Administration` tab -> `Environment-related` click `Environment`
2. Click "Add environment" button
3. Select Docker standalone, Start wizard
4. Select "Edge Agent Standard"
5. Set a name (the vm name)
6. "Portainer API server URL" should be `https://portainer.local.jabbas.dev`
7. "Portainer tunnel server address" should be `infrastrcture.machine.local.jabbas.dev:8000`
8. Click "Create"
