Steps:
- Part1
1. First ennable IOMMU in bios.

2. check in proxmox that IOMMU is enbalbed

3. block the GPU in proxmox: (this is for nvidia):
echo "blacklist nouveau" >> /etc/modprobe.d/blacklist.conf 
echo "blacklist nvidia*" >> /etc/modprobe.d/blacklist.conf

- Part2
1. Use ubuntu server with the minimal install.
2. after setup shutdown system and add grafic card to vm
RUN:
1. sudo apt update && sudo apt upgarde -y
2. ubuntu-drivers devices
4. sudo apt install <Recomended driver>
3. sudo apt-get install -y nvidia-container-toolkit
4. sudo apt install -y nvidia-docker2
5. nvidia-smi
